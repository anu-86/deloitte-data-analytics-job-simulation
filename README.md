## Task 1: Equality Score Analysis (Excel)

### Objective
Analyze employee compensation data and classify equality scores.

### Steps Performed
- Reviewed employee compensation data.
- Created an Equality Class column.
- Applied IF formulas to classify equality scores.
- Categorized records as Fair, Highly Discriminative, or Reverse Discriminative.
- Analyzed the results and documented findings.

### Skills Used
- Microsoft Excel
- IF Formulas
- Data Analysis
- Data Classification
